# Introduction
This project helps you read any wikipedia page in your linux terminal.

## Dependencies
* Lynx

## Setup 

#### Install
```
$ git clone https://github.com/vansh-goel/wiki-cli
$ cd wiki-cli
$ chmod +x wiki-cli
$ sudo cp wiki-cli.sh /usr/local/bin
```
#### Uninstall
```
$ cd /usr/local/bin
$ sudo rm wiki-cli
```
